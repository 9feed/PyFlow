  b = (DataTypes.Float, 0.0) 
)

Return atan(a / b), in radians.

The result is between -pi and pi. The vector in the plane from the origin to point (a, b) makes this angle with the positive X axis. The point of atan2() is that the signs of both inputs are known to it, so it can compute the correct quadrant for the angle. For example, atan(1) and atan2(1, 1) are both pi/4, but atan2(-1, -1) is -3*pi/4
